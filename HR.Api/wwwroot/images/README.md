# إضافة اللوجو للـ Payslip PDF

## الخطوات:

1. احفظ صورة اللوجو بتاعتك في المسار التالي:
   ```
   wwwroot/images/logo.png
   ```

2. تأكد إن الصورة:
   - بصيغة PNG عشان الخلفية الشفافة
   - الأبعاد المناسبة تقريباً: 550x120 بكسل
   - الخلفية زرقاء داكنة أو شفافة

3. لو عايز تغير حجم اللوجو في الـ PDF، عدل الأرقام دي في الكود:
   ```csharp
   row.ConstantItem(180).Height(50).Image(logoPath);
   ```
   - `180` = العرض
   - `50` = الارتفاع

## ملاحظات:
- لو الصورة مش موجودة، الـ PDF هيتعمل عادي بدون اللوجو
- اسم الملف لازم يكون `logo.png` بالظبط
- لو عايز تستخدم صيغة تانية (jpg مثلاً)، غير الامتداد في الكود

